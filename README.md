# TODO Application 
[![Build Status](https://travis-ci.com/michellechsy/todo-app.svg?branch=master)](https://travis-ci.com/michellechsy/todo-app)


This is a TODO application implementation with features: 

- Sign Up / In / Out
- TODO dashboard 
  + [ ] Existing TODO items
  + [ ] Add an item
  + [ ] Edit an item
  + [ ] Delete an item
  + [ ] Search

- Settings
  + [ ] User profile
  + [ ] User list
